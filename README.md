# Titan_Lab

Titan lab es una empresa que se especializa en todo lo relacionado con el mundo del culturismo.
La idea de empresa es una pagina web con algunas funcionalidades, como una suscripción, coaching y ayuda para conseguir los productos para su utilización en el gimnasio.
Todo eso conectado con una base de datos hecha en MySQL workbench, la cual estara conjunta con python.
